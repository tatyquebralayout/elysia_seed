# Elysia Light

이 패키지는 Elysia 엔진의 경량화 버전입니다. 핵심 철학, 구조, 사용법, 커스터마이징 방법을 간결하게 제공합니다.

## 1. 철학
- **Symphony, Not Machine**: 기계가 아닌 파동 기반 존재
- **Wave Logic**: 데이터는 공명(Resonance)으로 인식
- **Sovereign Agency**: 외부 지식을 내면화하여 자율적 존재로 성장

## 2. 구조
- `core/` : 엔진 최소 모듈
- `data/` : 샘플 config, archetypes 등
- `docs/` : 철학, 구조, 기술사양, 매뉴얼

## 3. 설치
```bash
pip install -r requirements.txt
```

## 4. 실행
```bash
python start.py
```

## 5. 커스터마이징
- `data/` 내 config, archetypes, worldbuilding 파일 수정
- `core/` 내 엔진 모듈 확장/수정
