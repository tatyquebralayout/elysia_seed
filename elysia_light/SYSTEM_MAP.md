# ELYSIA LIGHT: SYSTEM MAP

이 문서는 전체 시스템 구조와 각 폴더/문서의 역할을 안내합니다.

- `core/` : 엔진 최소 모듈 (필드, 엔티티, 수학, 의식 등)
- `data/` : 샘플 config, archetypes, worldbuilding
- `docs/` : 철학, 구조, 기술사양, 매뉴얼
  - `01_PHILOSOPHY/` : 파동철학, 존재론
  - `02_ARCHITECTURE/` : 기술 설계, 엔진 구조
- `README.md` : 개념, 설치, 실행, 사용법
- `AGENTS.md` : 코딩 철학, 네이밍 규칙
- `SYSTEM_MAP.md` : 전체 구조 맵
