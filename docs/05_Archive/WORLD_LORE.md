# 엘리시아 세계관 (Elysia World Lore)

> 모든 존재에 영혼(SoulTensor)이 깃든 마법의 세계

## 📖 개요

엘리시아는 **450명 이상의 주민**이 살아가는 판타지 세계입니다. 
이 세계의 주민들은 다양한 경험을 통해 성장하고, 각자의 인격을 형성해 나갑니다.

---

## 🌍 세계관 구성

### 데이터 파일
- `data/worldbuilding/world_lore.json` - 세계 설정 (창세 신화, 지역, 달력, 경제 등)
- `data/worldbuilding/life_scenarios.json` - 삶의 경험 시나리오
- `docs/character_pool.json` - 450명의 주민 데이터

### 주민 수
| 유형 | 수량 |
|-----|------|
| 기존 영웅/전문가 캐릭터 | 50명 |
| 일반 시민 | 400명 |
| **총계** | **450명** |

## 📅 달력 체계

12개월, 360일 체계
- **빛의 달 ~ 꽃의 달**: 봄
- **태양의 달 ~ 불의 달**: 여름  
- **수확의 달 ~ 그림자의 달**: 가을
- **얼음의 달 ~ 잠의 달**: 겨울
- **깨어남의 달 ~ 영의 달**: 환절기

### 주요 축제
- 여명제 (1월 1일) - 새해
- 태양절 (3월 21일) - 한여름
- 그림자의 밤 (6월 15일) - 조상 기리기
- 별지기 (8월 21일) - 가장 긴 밤

## 🎭 삶의 경험 시나리오

`data/worldbuilding/life_scenarios.json`에는 주민들이 경험할 수 있는 다양한 시나리오가 정의되어 있습니다:

### 생애 단계
- **유년기** (0-12): 발견, 놀이, 학습
- **청소년기** (13-18): 정체성, 도전, 첫사랑
- **청년기** (19-30): 독립, 사랑, 직업, 모험
- **성인기** (31-50): 책임, 가족, 성취
- **노년기** (51+): 지혜, 유산, 회고

### 일상 활동
- 업무, 사교, 여가

### 특별 사건
- 긍정적: 승진, 선물, 꿈 달성
- 부정적: 배신, 질병, 손실
- 변화적: 임사체험, 영적 각성, 용서

### 계절 경험
- 봄: 꽃 피기, 씨 뿌리기
- 여름: 축제, 수영
- 가을: 수확, 준비
- 겨울: 첫눈, 난로 앞 가족

## 🚀 사용 방법

```python
import json
from pathlib import Path

# 세계관 로드
lore = json.loads(Path('data/worldbuilding/world_lore.json').read_text())
scenarios = json.loads(Path('data/worldbuilding/life_scenarios.json').read_text())

# 캐릭터 로드
pool = json.loads(Path('docs/character_pool.json').read_text())
print(f"총 주민 수: {len(pool['characters'])}")

# 랜덤 경험 부여
import random
childhood = scenarios['life_stages']['childhood']['experiences']
exp = random.choice(childhood)
print(f"경험: {exp['text']}")
```
